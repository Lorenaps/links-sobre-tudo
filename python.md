  - Exemplo de uso:

    ~~~python
    remover_acentos = lambda s: unicodedata.normalize('NFKD',str(s)).encode('ASCII','ignore').decode('utf-8')
    
    s = 'óbvio'
    print(remover_acentos(s))
    # obvio
    
    ~~~
